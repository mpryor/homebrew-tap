# Mpryor Tap

## How do I install these formulae?

`brew install mpryor/tap/<formula>`

Or `brew tap mpryor/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mpryor/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
